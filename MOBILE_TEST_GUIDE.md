## 🔍 よくある問題と解決方法

### **問題1: プレーヤーがはみ出す**

**原因:** CSSの設定ミス

**確認:**
```css
.card-cover--spotify iframe {
  width: 100%; /* ← これが必須 */
}
```

**解決:** すでに設定済みなので問題なし

---

### **問題2: 1列にならない**

**原因:** グリッドの設定

**確認:**
```css
.grid {
  grid-template-columns: repeat(auto-fill, minmax(260px, 1fr));
}
```

**解決:** 画面幅が260px以上あれば2列、260px未満で1列になる

---

### **問題3: プレーヤーが正方形にならない**

**原因:** 高さの設定

**確認:**
```css
.card-cover--spotify {
  padding-top: 100%; /* ← これで1:1比率 */
}
```

**解決:** すでに設定済みなので問題なし

## ❓ よくある質問

### Q: スマホで実際に試せないけど大丈夫？

**A: 大丈夫です**
- Chrome/Edgeの開発者ツールで十分確認できます
- 実際のスマホとほぼ同じ表示になります
- タッチ操作もマウスでシミュレートできます

### Q: プレーヤーの高さを変えたい

**A: CSSを編集**
```css
.card-cover--spotify {
  padding-top: 100%; /* ← この値を変更 */
}
```
- 100% = 正方形（1:1）
- 75% = 横長（4:3）
- 56.25% = 超横長（16:9）

### Q: スマホで再生できない

**A: Spotifyアプリが必要**
- Spotifyアプリがインストールされていれば再生できます
- アプリがない場合はSpotifyへのリンクが表示されます
